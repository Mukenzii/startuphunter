┌─────────────────────────────────────────────────────────────────────────┐
│                         USER INTERACTION FLOW                            │
└─────────────────────────────────────────────────────────────────────────┘

1. Page Load
   └─→ main.jsx loads
       └─→ App.jsx renders
           └─→ All components render
               └─→ Mock data displays

2. Category Click
   └─→ User clicks category button
       └─→ Categories.jsx calls onCategoryChange()
           └─→ App.jsx updates activeCategory
               └─→ SectionsGrid filters sections
                   └─→ Only relevant cards show

3. Load More Click
   └─→ User clicks "Ko'proq muammolar"
       └─→ SectionsGrid updates visibleCount (+4)
           └─→ 4 more cards appear
               └─→ Button hides if all shown

4. Section Card Click
   └─→ User clicks a section card
       └─→ SectionCard calls onClick()
           └─→ SectionsGrid navigates to detail
               └─→ SectionDetail renders
                   └─→ Full content displays

5. Back Button Click
   └─→ User clicks back button
       └─→ SectionDetail navigates to home
           └─→ App.jsx renders
               └─→ Previous state restored
